# TruePeopleSearch

## Category
People OSINT / Public Records

## Purpose
TruePeopleSearch provides free access to public records including names, addresses, phone numbers, relatives, and associates for basic identity investigations.

## Access / Where to Get It
Website: https://www.truepeoplesearch.com  

## Input
Full name  
Phone number  
Address  

## Output
Current and past addresses  
Phone numbers  
Relatives  
Possible associates  
Age range  

## Strengths
Free access  
Fast lookups  
Good for basic identity validation  
No account required  
Useful for initial triage  

## Limitations
Manual searches only  
No official API  
Data accuracy varies  
US-focused  

## Requires Account
No  

## Typical Use Case
You are validating a name or phone number from a phishing report and need quick public-record context before escalating to paid tools.

## Example

Input:  
Jane Smith  

Output:

Location – Savannah, GA  
Phone – +1-912-XXX-XXXX  
Relatives – Robert Smith, Mary Smith  

This provides basic attribution and relationship context.

## Notes
Good starting point before paid tools.  
Often paired with Pipl or social media OSINT.  
Useful for quick verification.
