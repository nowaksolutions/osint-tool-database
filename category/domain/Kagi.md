# Kagi

## Category
Domain / Infrastructure Recon

## Purpose
Kagi is a privacy-focused search engine designed to provide high-quality, ad-free search results. It is used in OSINT investigations to research domains, organizations, exposed documents, and infrastructure without personalization bias or tracking.

Kagi is especially useful for clean reconnaissance when Google results become noisy or manipulated.

## Access / Where to Get It
Website: https://kagi.com  

## Input
Domain name  
Organization name  
Keywords  
Advanced search operators  

## Output
Search results  
Indexed documents  
Referenced domains  
Technical articles  
Public disclosures  

## Strengths
No ads or SEO spam  
Privacy-first search  
High-quality ranking  
Supports advanced operators  
Excellent alternative to Google for OSINT  

## Limitations
Requires paid subscription  
No official API  
Manual searching only  

## Requires Account
Yes  

## Typical Use Case
You are researching a domain or organization and want unbiased search results for exposed documents, breach reports, infrastructure references, or technical writeups.

## Example

Input:
example-company.com  

Output:

Public PDF documents  
Archived login portals  
Technical blog references  
Third-party service mentions  

This helps identify exposed assets and external dependencies.

## Notes
Excellent companion to DorkGPT and Dork Search Pro.  
Useful for clean recon without algorithmic personalization.  
Great for organizational footprint discovery.  
Best paired with DNSDumpster and SecurityTrails.
