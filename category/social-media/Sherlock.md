# Sherlock

## Category
Social Media OSINT / Username Enumeration

## Purpose
Sherlock is an open-source tool used to locate usernames across hundreds of social media platforms to identify digital footprints and linked accounts.

## Access / Where to Get It
GitHub: https://github.com/sherlock-project/sherlock  

## Input
Username  

## Output
List of platforms where the username exists  
Profile URLs  
Account status  

## Strengths
Completely free and open source  
Supports hundreds of platforms  
Fast enumeration  
Easy CLI usage  
Integrates well into OSINT workflows  

## Limitations
False positives possible  
Requires manual validation  
No official API  
Platform detection depends on site behavior  

## Requires Account
No  

## Typical Use Case
You receive a username from a breach or phishing campaign and want to identify which social platforms the user is active on.

## Example

Input:  
john_doe123  

Output:

Twitter – Active  
Instagram – Active  
Reddit – Active  
GitHub – Active
