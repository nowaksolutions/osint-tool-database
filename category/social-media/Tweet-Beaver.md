# TweetBeaver

## Category
Social Media OSINT / Twitter Analysis

## Purpose
TweetBeaver provides browser-based tools for analyzing Twitter/X accounts, including followers, following, engagement, and timeline activity.

## Access / Where to Get It
Website: https://tweetbeaver.com  

## Input
Twitter/X username  

## Output
Follower lists  
Following lists  
Tweet history  
Engagement metrics  
Account relationships  

## Strengths
Free web-based access  
No installation required  
Good for relationship mapping  
Fast timeline analysis  
Useful for influence tracking  

## Limitations
Limited to Twitter/X  
No API  
Manual usage only  
Dependent on platform availability  

## Requires Account
No  

## Typical Use Case
You are investigating a social engineering campaign and need to identify associated Twitter accounts and follower relationships.

## Example

Input:  
suspicious_user  

Output:

Followers – 1,240  
Following – 57  
Top Interactions – user1, user2  

This helps identify connected accounts and influence patterns.

## Notes
Great for social graph analysis.  
Pairs well with Sherlock and Maltego.  
Useful for tracking coordinated activity.
