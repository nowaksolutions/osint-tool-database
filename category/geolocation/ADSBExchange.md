# ADSBexchange

## Category
Geolocation / Physical Asset Intelligence

## Purpose
ADSBexchange provides real-time and historical aircraft tracking using ADS-B broadcasts, allowing investigators to monitor aircraft movements, identify ownership patterns, and correlate aviation activity with events or locations.

Unlike commercial flight trackers, ADSBexchange does not censor government, military, or private aircraft.

## Access / Where to Get It
Website: https://www.adsbexchange.com  
Live Map: https://globe.adsbexchange.com  

## Input
Aircraft hex ID  
Tail number  
Callsign  
Geographic area  

## Output
Real-time aircraft location  
Altitude and speed  
Flight paths  
Historical tracks  
Aircraft type  
Registration information
