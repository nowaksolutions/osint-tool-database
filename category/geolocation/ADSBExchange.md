# ADSBexchange

## Category
Geolocation / Physical Asset Intelligence

## Purpose
ADSBexchange provides real-time and historical aircraft tracking using ADS-B broadcasts, allowing investigators to monitor aircraft movements, identify ownership patterns, and correlate aviation activity with events or locations.

Unlike commercial flight trackers, ADSBexchange does not censor government, military, or private aircraft.

## Access / Where to Get It
Website: https://www.adsbexchange.com  
Live Map: https://globe.adsbexchange.com  

## Input
Aircraft hex ID  
Tail number  
Callsign  
Geographic area  

## Output
Real-time aircraft location  
Altitude and speed  
Flight paths  
Historical tracks  
Aircraft type  
Registration information  

## Strengths
Unfiltered global ADS-B data  
Tracks military and private aircraft  
Real-time visualization  
Historical playback  
No account required  

## Limitations
No official API  
Manual analysis  
Data dependent on receiver coverage  
Aircraft ownership may be obfuscated  

## Requires Account
No  

## Typical Use Case
You are investigating activity near a sensitive location or major event and want to determine whether unusual aircraft movements occurred during a specific timeframe.

## Example

Input:  
Geographic area – Savannah, GA  

Output:

Aircraft – Cessna 208  
Altitude – 6,500 ft  
Callsign – N123AB  
Flight Path – Repeated circular pattern  

This may indicate aerial surveying, training operations, or law enforcement activity.

## Notes
Useful for adding physical-world context to investigations.  
Often used by journalists and OSINT analysts.  
Can support situational awareness during incidents.  
Best paired with geolocation and infrastructure tools.
