# WhatsMyName

## Category
Username Enumeration / Identity Discovery

## Purpose
WhatsMyName is an open-source username enumeration tool used to discover accounts across hundreds of websites by checking for the existence of a specific username.

## Access / Where to Get It
GitHub: https://github.com/WebBreacher/WhatsMyName  

## Input
Username  

## Output
List of platforms where the username exists  
Profile URLs  
Confidence indicators  

## Strengths
Open source  
Large curated site database  
JSON output supported  
Integrates with automation pipelines  
Maintained by OSINT professionals  

## Limitations
False positives possible  
Requires manual verification  
CLI-based  
No official API  

## Requires Account
No  

## Typical Use Case
You obtain a username from breach data or phishing artifacts and want to determine which platforms the individual uses.

## Example

Input:  
shadowwolf  

Output:

GitHub – Active  
Reddit – Active  
Instagram – Active  

This establishes a digital footprint and supports identity correlation.

## Notes
Widely used in OSINT investigations.  
Pairs well with Sherlock.  
Great starting point for online persona mapping.
