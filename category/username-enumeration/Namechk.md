# Namechk

## Category
Username Enumeration / Availability Checking

## Purpose
Namechk checks username availability and existence across social media platforms and domain extensions to identify active accounts or unused names.

## Access / Where to Get It
Website: https://namechk.com  

## Input
Username  

## Output
Platform availability status  
Existing accounts  
Available domains  

## Strengths
Simple web interface  
Fast results  
No installation required  
Good for quick validation  

## Limitations
Manual only  
No API  
Limited platform depth  
No historical context  

## Requires Account
No  

## Typical Use Case
You need a quick check to see whether a username already exists on common platforms before deeper OSINT analysis.

## Example

Input:  
nightfox  

Output:

Twitter – Taken  
Instagram – Taken  
GitHub – Available  

This helps determine where to pivot next.

## Notes
Best for quick surface checks.  
Often used before Sherlock or WhatsMyName.  
Useful during early investigation stages.
